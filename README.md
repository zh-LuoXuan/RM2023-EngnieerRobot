# RM2023-EngnieerRobot
相比于Robomaster2022赛季，工程机器人的基本职能被裁剪，着重强化了机器人的抓取与兑换功能。针对这一改动，有两个开发的大方向，一是维持框架伸缩变形结构，二是做机械臂结构。经过激烈的讨论之后，最终确定维持框架结构，而对此我们做了以下几点尝试：
1.由2022赛季的工程直接改造得到新工程，即：https://github.com/zh-LuoXuan/RM2023-EngnieerRobot/tree/main/2022%E5%B7%A5%E7%A8%8B%E6%94%B9%E9%80%A0%E7%89%88%E6%9C%AC
（2022工程改造版本），改造得到的新机作为验证机确定了延用伸缩变形结构可行，但是直接改造得到是不适用于比赛的，还得造新的机器人。
2.设计2023分区赛工程，由于新规则限制，我们舍弃了夹爪结构而全程采用吸盘获取与兑换矿石，同时增加了翻转矿石机构用于将矿石翻转到正确的姿态，具体电控代码详见：https://github.com/zh-LuoXuan/RM2023-EngnieerRobot/tree/main/2023%E5%88%86%E5%8C%BA%E8%B5%9B%E7%89%88%E6%9C%AC
（2023分区赛版本）。
3.在分区赛之后，我们积极备战复活赛国赛，由分区赛表现得出结论：原有的设计是有很大问题的，基于已经发现的问题，我们从零开始设计新的结构与框架，将夹取与兑换结构分开设计，取消原有的矿石翻转机构等等，由此得到了新的复活赛国赛版本工程机器人，具体电控代码详见：https://github.com/zh-LuoXuan/RM2023-EngnieerRobot/tree/main/2023%E5%A4%8D%E6%B4%BB%E8%B5%9B%E5%9B%BD%E8%B5%9B%E7%89%88%E6%9C%AC
（2023复活赛国赛版本）。
